# 싱글섯다 (Single Hwatu)

## 1. 게임이름 - 정보
**섯다** - 한국 전통 카드게임을 콘솔 환경에서 구현한 게임입니다. 화투패를 이용한 전통적인 섯다 게임의 규칙을 따르며, 텍스트 기반의 인터페이스로 플레이할 수 있습니다.

## 2. 사용툴
- **언어**: C#
- **개발환경**: Console Application
- **IDE**: Visual Studio

## 3. 기간
2024년

## 4. 개발인원
1명 (개인 프로젝트)

## 5. 플랫폼
- Windows Console
- .NET Framework

## 6. 담당
**Full Stack Development**
- 게임 로직 구현
- 카드 UI 디자인 (ASCII 아트)
- 사용자 인터페이스 개발
- 전체 시스템 설계 및 구현
